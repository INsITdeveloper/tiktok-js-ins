#### usage (CLI)
```
usage: node index.js [-h] [-u] [-a] [-m] [-v] [-g] [-e]

A simple tiktok downloader @ motebaya 2022

optional arguments:
  -h, --help     show this help message and exit
  -u , --url     url tiktok video
  -a, --avatar   only get avatar
  -m, --music    url get music
  -v, --video    only get tiktok video (without watermark)
  -g, --grab     download all media, include music, video .etc

additional:
  if u dont know to usage, just use this command

  -e, --example  show example for usage
```
